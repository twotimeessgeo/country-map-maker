# 투타임즈의 통계머신

선택한 국가나 한국 권역만 색칠해서 지역 지도를 만들고, 현재 보이는 결과를 그대로 SVG로 내보내는 작은 웹 앱입니다. `index.html`을 직접 열어도 동작하도록 로컬 라이브러리와 데이터가 함께 들어 있습니다.

## 사용 방법

1. [index.html](/Users/twotimess/Desktop/Map/index.html)를 브라우저에서 바로 열거나, 터미널에서 `npm run start` 후 [http://localhost:4173](http://localhost:4173) 로 접속합니다.
2. 세계 지도에서는 검색창에 영문 국가명을 입력하거나 지도를 클릭해서 국가를 추가합니다.
3. 한국 지도에서는 상단 `대한민국` 탭으로 전환한 뒤 `도/광역시`, `시/군`, `구/군` 권역을 칩이나 지도 클릭으로 켜고 끕니다. 일반 시의 `구`는 `시/군` 단계에서 합쳐서 보이고, 서울·광역시는 `구/군` 단계에서 별도로 볼 수 있습니다.
4. 한국 권역은 선택 목록에서 여러 개를 체크한 뒤 `체크한 권역 비교`를 누르면 전국 지도에서 함께 비교할 수 있습니다. 부모 범위가 달라도 한 장에서 같이 볼 수 있고, 필요하면 노선 오버레이도 덧씌울 수 있습니다.
5. `시/군` 단계는 전국 지도를 기본으로 보되, `경기도 + 강원특별자치도`처럼 여러 도/광역시 범위를 고르면 그 범위만 남긴 시/군 지도로 바로 볼 수 있습니다.
6. 각 국가/권역의 색상을 바꾸고, 캔버스 크기와 여백을 조절합니다. 미리보기 오른쪽 아래 핸들로도 파워포인트 도형처럼 비율을 바로 바꿀 수 있습니다.
7. 세계 지도에서는 프리뷰 왼쪽 `Quick Workspace`에서 `영역 확대`, `이동`, `마커 추가`, `인셋 추가`를 전환할 수 있고, `1-4` 키로도 빠르게 바뀝니다. 한국 지도에서는 이동 대신 권역 on/off 중심의 고정 보기로 전환됩니다.
8. 세계 평면 지도는 고위도 왜곡을 줄인 밀러 계열 평면으로 보고, 북극/남극 시점은 극점을 가운데에 고정한 채 회전합니다.
9. 세계 지도는 드래그 박스 확대, 휠/핀치 줌, `Space` 임시 이동을 지원하며 확대 수준에 따라 `110m / 50m / 10m` 해안선 디테일이 자동으로 바뀝니다.
10. 적도, `30°N/S`, `60°N/S`, 북회귀선, 남회귀선을 켜고, 선은 끄고 단위 라벨만 남길 수도 있습니다.
11. 점, 원형, 점선 타원형, 검은 점 마커를 클릭 또는 드래그로 추가하고, 미리보기 위 편집 박스에서 바로 옮기거나 크기를 조절할 수 있습니다.
12. 인셋은 드래그 박스로 만들고, 생성 뒤에는 미리보기 위 패널을 직접 끌거나 리사이즈해 위치를 다듬습니다.
13. 국경선/행정 경계선은 `실선 / 점선 / 끄기`로 바꿀 수 있습니다.
14. `SidaeAi_S` 글꼴, 글자 크기, 축척 막대 표시 여부를 조절합니다.
15. 실수했을 때는 `실행 취소`/`다시 실행` 또는 `Cmd/Ctrl+Z`, `Shift+Cmd/Ctrl+Z`를 사용할 수 있습니다.
16. `SVG 내보내기`를 누르면 현재 미리보기가 SVG 파일로 저장됩니다.

## 출제형 그래프

- 세계 지도 `2. DETAIL` 영역의 `출제형 그래프` 패널에서 시험지 스타일 그래프를 바로 만들 수 있습니다.
- 프리셋은 `구성 누적막대`, `단일 순위막대`, `짝막대 비교`, `2시점 비교`, `시계열 선그래프`, `산포도`, `상위 3개국+기타` 일곱 가지입니다.
- 그래프 종류에 따라 `양`, `비율`, `상댓값100` 표시 방식을 바로 바꿀 수 있고, 구성형 그래프는 `실제 양`과 `100 환산 비율`을 모두 지원합니다.
- `가명 처리`를 켜면 `(가)`, `(나)` 식으로 라벨을 바꾼 그래프와 함께 정답표가 같이 만들어집니다.
- 선택 국가가 있으면 그 국가를 우선 쓰고, 없으면 전체 자료에서 상위 국가나 대륙을 자동으로 골라 그래프를 생성합니다.
- `국가 랜덤`, `통계 랜덤`, `세트 랜덤` 버튼으로 출제에 잘 맞는 국가 묶음, 통계 주제, 그래프 조합을 바로 추천할 수 있습니다.
- 랜덤 추천은 상위권 국가만 단순 추출하지 않고, 대륙 분산, 대비가 큰 값, 시계열 변화폭, 사분면 분포처럼 문항으로 쓰기 좋은 차이를 우선 반영합니다.
- 그래프는 흑백 패턴 SVG로 렌더링되며, 패널 안의 `SVG 내보내기` 버튼으로 그래프만 따로 저장할 수 있습니다.
- 축, 프레임, 보조선, 범례, 채우기 패턴은 공통 `출제형 흑백 테마`를 사용합니다. 100% 그래프는 `0·20·40·60·80·100(%)` 축을 우선 쓰고, 실제 양 그래프는 같은 프레임과 점선 보조선 규칙으로 정리됩니다.
- 그래프 SVG의 기준 너비는 최대 `310px`이며, 모든 배치 간격은 2px 그리드에 맞춥니다. 제목 `9pt`, 라벨/범례 `8pt`, 축 눈금 `7.5pt`, 주석 `7pt`를 `SidaeAi_S` 장평 95%로 쓰고, 선과 윤곽선은 기본 `0.4pt`로 반 픽셀 좌표에 스냅합니다.
- 그래프-범례 간격은 `10px`, 범례-주석 간격은 `8px`, 블록 간격은 `12px`입니다. 범례 항목은 4개를 넘으면 다음 줄로 넘기고, 미리보기 패널 폭이 `92px` 아래로 내려가면 다중 패널을 세로로 쌓습니다.
- 채우기는 `연회색 → 진회색 → 흰색 → 중회색 → 세로줄 → 사선 → 점 → 가로줄` 순서를 기본으로 하며, 복사본에서도 구분되도록 모든 막대 조각에 검정 외곽선을 둡니다.
- 오른쪽에는 값 정리, 가명 정답표, `출제 확장` 카드가 함께 표시되어 어떤 보조 통계를 추가하면 좋은지도 바로 확인할 수 있습니다.

## 한국지리 통계 패널

- 한국 지도 `2. DETAIL` 영역의 공식 통계 패널에서도 `현재 범위 사용`, `지역 랜덤`, `통계 랜덤`, `세트 랜덤` 버튼을 바로 쓸 수 있습니다.
- `지역 랜덤`은 현재 지표를 기준으로 비교 포인트가 살아 있는 시도·시군을 골라 주고, `통계 랜덤`은 현재 보기 방식에 맞는 지표나 산포도 축 조합을 추천합니다.
- `세트 랜덤`은 대분류, 보기 방식, 지표, 추천 지역을 함께 바꿔서 한국지리 문항용 비교 자료를 빠르게 찾도록 돕습니다.
- 한국지리 막대·누적막대·선그래프·산포도도 출제형 그래프와 같은 `0.4pt` 선 두께와 `3~5px` 점 크기 기준을 씁니다.

- 세계/대한민국 버전 전환

- 경부고속도로, 경부고속철도, 영동고속도로, 호남고속철도 단순화 오버레이
- `110m / 50m / 10m` 다중 세계 경계 데이터 기반 자동 디테일 전환
- 밀러 계열 평면 지도와 북극/남극 시점 전환
- 평면 지도 좌우 연속 렌더링
- 중심 경도 조절과 세계 중심 이동
- 캔버스 비율과 크기 조절
- 미리보기 코너 핸들 기반 비율 조절
- 프리뷰 옆 `Quick Workspace` 빠른 도구 패널
- 세계지리 출제형 흑백 패턴 그래프 프리셋 7종
- 그래프별 `양 / 비율 / 상댓값100` 표시 방식 전환
- 선택 국가 우선 / 자동 상위 국가·대륙 추출 기반 시험지형 그래프 생성
- 유의미한 국가 랜덤 / 통계 랜덤 / 세트 랜덤 추천
- 한국지리 공식 통계 패널의 유의미한 지역 랜덤 / 통계 랜덤 / 세트 랜덤 추천
- 가명 라벨 `(가)`, `(나)` 처리와 정답표 자동 생성
- 그래프 전용 SVG 내보내기와 출제 확장 포인트 안내
- `실행 취소` / `다시 실행` 버튼과 단축키
- 드래그 박스 확대, 팬, 휠/핀치 줌, 줌 아웃, 전체 보기
- `1-4`, `0`, `Space`, `Cmd/Ctrl+Z` 기반 빠른 보기 도구 단축키
- 위선 표시 옵션: 적도, `30°N/S`, `60°N/S`, 북회귀선, 남회귀선
- 기준선 선 숨김과 단위 라벨만 표시

- 로컬 라이브러리와 데이터 포함
