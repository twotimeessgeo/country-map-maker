# Country Map Maker

선택한 국가만 색칠해서 자동으로 확대된 지역 지도를 만들고, 현재 보이는 결과를 그대로 SVG로 내보내는 작은 웹 앱입니다. `index.html`을 직접 열어도 동작하도록 로컬 라이브러리와 데이터가 함께 들어 있습니다.

## 사용 방법

1. [index.html](/Users/twotimess/Desktop/Map/index.html)를 브라우저에서 바로 열거나, 터미널에서 `npm run start` 후 [http://localhost:4173](http://localhost:4173) 로 접속합니다.
2. 검색창에 영문 국가명을 입력하거나 지도를 클릭해서 국가를 추가합니다.
3. 각 국가의 색상을 바꾸고, 캔버스 크기와 여백을 조절합니다. 미리보기 오른쪽 아래 핸들로도 파워포인트 도형처럼 비율을 바로 바꿀 수 있습니다.
4. 보기 도구는 중복 카드 없이 프리뷰 왼쪽 `Quick Workspace` 하나로 정리되어 있습니다. 여기서 `영역 확대`, `이동`, `마커 추가`, `인셋 추가`를 바로 전환할 수 있고, `1-4` 키로도 빠르게 바뀝니다.
5. 기본 평면 지도는 고위도 왜곡을 줄인 밀러 계열 평면으로 보고, 북극/남극 시점은 극점을 가운데에 고정한 채 회전합니다.
6. 미리보기에서 드래그 박스로 확대하거나, 휠과 핀치로 바로 줌 인·아웃합니다. 확대 모드에서는 짧게 클릭해 국가를 바로 선택할 수 있고, `Space`를 누르는 동안은 임시로 이동 모드가 됩니다. 확대 수준에 따라 `110m / 50m / 10m` 해안선 디테일이 자동으로 바뀝니다.
7. 적도, `30°N/S`, `60°N/S`, 북회귀선, 남회귀선을 켜고, 선은 끄고 단위 라벨만 남길 수도 있습니다.
8. 점, 원형, 점선 타원형, 검은 점 마커를 클릭 또는 드래그로 추가하고, 미리보기 위 편집 박스에서 바로 옮기거나 크기를 조절할 수 있습니다.
9. 인셋은 드래그 박스로 만들고, 생성 뒤에는 미리보기 위 패널을 직접 끌거나 리사이즈해 위치를 다듬습니다.
10. 국경선은 `실선 / 점선 / 끄기`로 바꿀 수 있고, 평면 세계지도는 좌우로 이어진 상태로 렌더됩니다.
11. `SidaeAi_S` 글꼴, 글자 크기, 축척 막대 표시 여부를 조절합니다.
12. 실수했을 때는 `실행 취소`/`다시 실행` 또는 `Cmd/Ctrl+Z`, `Shift+Cmd/Ctrl+Z`를 사용할 수 있습니다.
13. `SVG 내보내기`를 누르면 현재 미리보기가 SVG 파일로 저장됩니다.

## 포함된 기능

- 선택 국가 기준 자동 확대
- `110m / 50m / 10m` 다중 세계 경계 데이터 기반 자동 디테일 전환
- 밀러 계열 평면 지도와 북극/남극 시점 전환
- 평면 지도 좌우 연속 렌더링
- 중심 경도 조절과 세계 중심 이동
- 캔버스 비율과 크기 조절
- 미리보기 코너 핸들 기반 비율 조절
- 프리뷰 옆 `Quick Workspace` 빠른 도구 패널
- `실행 취소` / `다시 실행` 버튼과 단축키
- 드래그 박스 확대, 팬, 휠/핀치 줌, 줌 아웃, 전체 보기
- `1-4`, `0`, `Space`, `Cmd/Ctrl+Z` 기반 빠른 보기 도구 단축키
- 위선 표시 옵션: 적도, `30°N/S`, `60°N/S`, 북회귀선, 남회귀선
- 기준선 선 숨김과 단위 라벨만 표시
- 국경선 `실선 / 점선 / 끄기` 옵션
- 점, 원형, 점선 타원, 검은 점 마커 추가와 미리보기 직접 이동/리사이즈
- 드래그 기반 인셋 확대도와 미리보기 직접 이동/리사이즈
- 축척 막대 표시 옵션
- `SidaeAi_S` 글꼴과 지도 글자 크기 조절
- 선택 국가 외 국경선 숨김 옵션
- 국가별 색상 지정
- 평면 지도는 바깥 프레임, 극 시점은 원형 윤곽선을 사용
- 모든 윤곽선 두께는 `0.4pt`로 유지
- 클릭 기반 추가/제거
- 로컬 라이브러리와 데이터 포함
